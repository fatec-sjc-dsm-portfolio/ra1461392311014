Sobre mim

Olá! Meu nome é Wellington Luiz de Faria. Entrei no mundo do Desenvolvimento de Software em 2019 e desde então sigo aprendendo e colocando em prática tudo o que estudo. Hoje sou concluinte de Desenvolvimento de Software Multiplataforma na Fatec São José dos Campos – Prof. Jessen Vidal e também atuo como Desenvolvedor Júnior.

Ao longo dessa jornada, já trabalhei com back-end, full-stack, DevOps e até sistemas embarcados, sempre buscando evoluir e me desafiar em diferentes contextos. Também tive a chance de assumir o papel de Scrum Master, o que me ajudou a desenvolver ainda mais minha organização e trabalho em equipe.

Aqui estão algumas das tecnologias, metodologias e conceitos com os quais tenho sólida experiência:

  • Desenvolvimento back-end com Python (Django, FastAPI, Flask) e Java (Spring Boot)
  • Desenvolvimento front-end com React, Next.js, TypeScript
  • Comunicação entre sistemas embarcados e APIs web (REST, WebSockets)
  • Manipulação e visualização de dados (Pandas, ChartJS, HighCharts)
  • Testes de integração, carga e performance (load tests)
  • Monitoramento de dados e análise de logs
  • Containerização e infraestrutura com Docker
  • Controle de versão e colaboração com Git
  • CI/CD e deploy de aplicações
  • Metodologias ágeis (Scrum) e planejamento de sprints
  • Trabalho colaborativo em equipe e resolução de problemas
  • Desenvolvimento de interfaces responsivas e UX
